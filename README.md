# Portfolio Site

## 🎯 概要

未経験からAIエンジニアを目指すポートフォリオサイトです。  
HTML / CSS / JavaScript のみで構成したシングルページサイトで、自己紹介・職務経歴・スキルセット・制作実績・趣味を掲載しています。

---

## 🎓 目的

- エンジニア転向の意思と経緯を伝えるポートフォリオとして制作
- HTML / CSS / JavaScript の基礎力を実践的に習得・証明する
- 採用担当者・クライアントへの第一印象となる「名刺」として活用する

## ✨ 工夫した点

### 明るくメリハリのあるデザイン
アクセントカラーにサンシャインイエロー（`#F2C12E`）を採用し、見やすさと印象を両立。  
CSS カスタムプロパティでカラー・フォント・イージングを一元管理し、可変フォント（Fraunces）の `opsz` 軸をサイズごとに調整することで見出しの字形を最適化しています。

### 地元の写真で個性を出す
ヒーローセクションの背景には自分で撮影した地元の風景写真を使用。  
既製素材ではなく自身の視点で切り取った写真を採用し、個性と温度感を持たせました。

### スキルセットの視覚的なわかりやすさ
スキルを「実務レベル」「調べながら作業可能」の2段階に分け、バー表示でレベル感を直感的に伝えています。  
バーは `IntersectionObserver` でスクロールに連動し、`transition-delay` で波状にアニメーションします。

### 切り絵セクションで人柄を伝える
スキル・実績だけでは伝わりにくい人となりを補足するため、趣味の切り絵作品をギャラリー形式で掲載。  
繊細な手仕事への集中力がエンジニアとしての姿勢にも通じると考えています。

### ポートフォリオサイト自体のスクリーンショット
本サイトの PC 版・スマホ版のスクリーンショットを Works セクションに掲載しています。

### 制作実績：Playout Pro（デモ画像・デモ動画）
業務で携わった案件のスクリーンショットと画面収録動画を掲載しています。

### PC・スマホ両対応のレスポンシブデザイン
メディアクエリを細かく設定し、PC・タブレット・スマホのいずれの画面幅でも文字サイズやレイアウトに違和感が生じないよう逐一調整しています。

## 📁 ディレクトリ構成

```
demosite/
├── index.html              # メインHTML（全セクション）
├── .gitignore
├── README.md
└── assets/
    ├── css/
    │   └── style.css       # 全スタイル（変数・レイアウト・アニメーション・レスポンシブ）

    │   └── script.js       # インタラクション（ナビ・スクロール・アコーディオン・パララックス）

        ├── about/          # プロフィール画像
```

## 🛠️ 技術スタック

| カテゴリ | 技術 |
|---|---|
| マークアップ | HTML5 |
| スタイリング | CSS3（カスタムプロパティ・Grid・Flexbox・アニメーション） |

| バージョン管理 | Git / GitHub |
