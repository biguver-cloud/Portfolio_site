# Portfolio Site

## 🎯 概要

未経験からAIエンジニアを目指すポートフォリオサイトです。  
HTML / CSS / JavaScript のみで構成したシングルページサイトで、自己紹介・職務経歴・スキルセット・制作実績・趣味を掲載しています。

---

## 🎓 目的

- エンジニア転向の意思と経緯を伝えるポートフォリオとして制作
- HTML / CSS / JavaScript の基礎力を実践的に習得・証明する
- 採用担当者・クライアントへの第一印象となる「名刺」として活用する

---

## ✨ 工夫した点

### 明るくメリハリのあるデザイン
「見やすい・読みやすい・印象に残る」を意識し、アクセントカラーにサンシャインイエロー（`#F2C12E`）を採用。  
各セクションの文言はシンプルかつ端的にまとめ、情報を詰め込みすぎないよう心がけました。  
技術的には **CSS カスタムプロパティ**でカラー・フォント・イージングを一元管理し、**可変フォント（Fraunces）** の `opsz` 軸をサイズごとに切り替えることで、大小どの見出しも最適な字形になるよう調整しています。

### 地元の写真で個性を出す
ヒーローセクションの背景には自分で撮影した地元の写真を使用しています。  
既製の素材ではなく自分の視点で切り取った風景を使うことで、ポートフォリオに個性と温度感を持たせたいと考えました。

### スキルセットの視覚的なわかりやすさ
スキルを「実務レベル」「調べながら作業可能」の2段階に分け、バー表示で一目でレベル感が伝わるようにしています。  
バーの伸長は `IntersectionObserver` でスクロールに連動して発火し、`transition-delay` でスキルごとに波状にずらすことで視覚的なリズムを演出しています。

### 切り絵セクションで人柄を伝える
スキルや実績だけでは伝わりにくい「人となり」を別の角度から知ってもらいたいと考え、趣味の切り絵作品をギャラリー形式で掲載しました。  
繊細な手仕事への集中力や、ものづくりへのこだわりがエンジニアとしての姿勢にも通じると感じています。

## 📁 ディレクトリ構成

```
Portfolio_site/
├── index.html              # メインHTML（全セクション）
├── .gitignore
├── README.md
└── assets/
    ├── css/
    │   └── style.css       # 全スタイル（変数・レイアウト・アニメーション・レスポンシブ）
    ├── js/
    │   └── script.js       # インタラクション（ナビ・スクロール・アコーディオン・パララックス）
    └── images/
        ├── about/          # プロフィール写真（GitHubには非公開）
        ├── hero/           # ヒーロー背景画像
        ├── hobby/          # 切り絵作品（著作権作品はGitHubに非公開）
        └── works/          # 制作物のスクリーンショット
```

## 🛠️ 技術スタック

| カテゴリ | 技術 |
|---|---|
| マークアップ | HTML5 |
| スタイリング | CSS3（カスタムプロパティ・Grid・Flexbox・アニメーション） |
| スクリプト | JavaScript（Vanilla ES6+） |
| フォント | Google Fonts（Fraunces / Noto Sans JP / Noto Serif JP） |
| アニメーション | IntersectionObserver API / requestAnimationFrame |
| ビルドツール | なし（ビルド不要） |
| バージョン管理 | Git / GitHub |
